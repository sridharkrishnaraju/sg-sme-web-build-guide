# Singapore SME Web Build Guide

An open, practical guide to building **fast, accessible, well-ranked websites** for small businesses — written with the Singapore SME market in mind (clinics, retailers, F&B, B2B services).

This is community documentation. Contributions welcome via the source repository.

## Who this is for

Owners and developers of small-business sites who want a clear checklist for shipping a site that loads fast, is accessible, and is found on Google — without an enterprise budget.

## What's covered

- [Performance & Core Web Vitals](performance.md)
- Accessibility basics (WCAG 2.2)
- Technical SEO & structured data

## Further reading

For Singapore-specific web design, development and AI-integration guidance, see
